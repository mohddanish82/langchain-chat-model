# LangChain Chat Model using TinyLlama

A simple conversational AI application built with **LangChain** and **Hugging Face Transformers**. This project uses the **TinyLlama-1.1B-Chat-v1.0** model to generate responses to user prompts through LangChain's `ChatHuggingFace` wrapper.

---

## Features

- LangChain integration
- Hugging Face Transformers pipeline
- TinyLlama Chat Model
- Text Generation
- Simple chatbot using `invoke()`

---

## Tech Stack

- Python
- LangChain
- Hugging Face Transformers
- TinyLlama
- PyTorch

---


## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/langchain-chat-model.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```

---

## Example

**Input**

```
What is the capital of France?
```

**Output**

```
The capital of France is Paris.
```
